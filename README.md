# Collection Decision Tree
Collection Decision Tree I created years ago for our bachelor and master students so they do not make the same mistakes again and again. Still relevant for checking that the LLM makes the right decision for you :)

## Render command
```bash
d2 -w --scale 0.8 --layout elk --elk-nodeNodeBetweenLayers 20 '.\Collection Decision Tree.d2' ".\www\collections-decision-tree\Collection Decision Tree.svg"
```

## Issues
Do not hesitate to report issues or file PRs, especially if some guidance seems incorrect or outdated. However, I have no ambition to make the collection decision tree complete (there are probably tons of collections out there that are not covered by this tree).
